## 📈 Excel

En este apartado, Excel fue relativamente sencillo para mí, ya que
cuento con conocimientos previos en el uso de la herramienta. Sin
embargo, quiero aclarar que las fórmulas que mencionaré a continuación
**no son exactamente las que utilicé en la actividad**, sino ejemplos de
referencia para que se entienda cómo funcionan y cómo pueden aplicarse.

### 📌 Fórmulas de referencia

``` excel
=SUMA(A5:A16)
```

Esta fórmula permite sumar un rango de celdas, desde la A5 hasta la A16.
Es muy útil cuando se tienen múltiples valores y se necesita una
sumatoria automática.

``` excel
=(B5-C5)*$E$2
```

En este ejemplo se realiza primero una resta y luego una
multiplicación.\
El símbolo `$` se utiliza para **bloquear una celda** (referencia
absoluta), lo que evita que cambie al momento de extender la fórmula
hacia otras filas o columnas.

``` excel
=B5+C5
```

Suma directa entre dos celdas.

``` excel
=B5-C5
```

Resta directa entre dos celdas.

``` excel
=D8*B8
```

Multiplicación entre dos celdas.

Estas fórmulas son ejemplos básicos que ayudan a entender cómo
estructurar operaciones dentro de Excel.


### 🔄 Replicar fórmulas automáticamente

Algo importante que tal vez no todos conocen es que, después de escribir
una fórmula, se puede posicionar el cursor en la esquina inferior
derecha de la celda hasta que aparezca el símbolo `+`.

Si se mantiene presionado el clic izquierdo y se arrastra hacia abajo (o
hacia un lado), Excel replicará la fórmula automáticamente, ajustando
las referencias según corresponda (a menos que estén bloqueadas con
`$`).

Esto facilita mucho el trabajo cuando se manejan tablas con varias
filas.
