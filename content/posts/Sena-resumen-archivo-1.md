# 📌 Introducción a las TIC

El estudio de la informática y las Tecnologías de la Información y la Comunicación (TIC) es fundamental para gestionar información y datos en formato digital de manera automática. Estas herramientas son indispensables para optimizar actividades tanto en la vida diaria como en el ámbito laboral y académico.

---

# 💻 Software y su Clasificación

El software es el conjunto de componentes lógicos que permiten realizar tareas específicas en un sistema computacional. Se clasifica principalmente en:

* **Software de Sistema:** Programas que administran los recursos del hardware, como los sistemas operativos (Windows, macOS, Android).
* **Software de Aplicación:** Herramientas diseñadas para tareas puntuales, como procesadores de texto, hojas de cálculo o navegadores.
* **Software de Programación:** Entornos y lenguajes que permiten crear otros programas (IDE como VS Code, compiladores).

---

# 🌐 Evolución y Uso del Internet

Internet es un ecosistema global de redes interconectadas que facilita el intercambio de información. 

### Hitos y conceptos clave:
* **Origen:** Evolucionó desde ARPANET hacia la World Wide Web (WWW).
* **Servicios:** Permite el uso de correo electrónico, transferencia de archivos y acceso a la nube.
* **Actualidad:** Se enfoca en la interconectividad masiva a través del 5G y el Internet de las Cosas (IoT).

---

# 📂 Herramientas de Ofimática

La ofimática comprende las técnicas y herramientas informáticas que se utilizan para optimizar y automatizar las tareas de oficina.

* **Procesadores de Texto:** Creación de documentos profesionales (ej. Microsoft Word).
* **Hojas de Cálculo:** Gestión de datos y fórmulas matemáticas (ej. Microsoft Excel).
* **Presentaciones:** Exposición visual de ideas (ej. PowerPoint).
* **Trabajo Colaborativo:** Plataformas que permiten a múltiples usuarios editar archivos en tiempo real, facilitando la productividad grupal.

---

# 📈 Mejora Continua

La aplicación de las TIC permite recolectar datos y generar estadísticas para la toma de decisiones. La **mejora continua** se basa en el análisis de estos resultados para optimizar procesos, reducir costos y elevar la calidad de los productos o servicios finales.
